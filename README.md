# NLW eSports

## Evento da Rocketseat de Setembro/2022

### Rodando o app mobile

1. cd mobile
2. npm install
3. expo start --localhost --android ou apenas expo start --android

### Rodando o web client

1. cd web
2. npm install
3. npm run dev

### Rodando o servidor

1. cd server
2. npm install
3. npm run dev
